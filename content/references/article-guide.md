# 아티클 작성 가이드라인

## 기본 원칙

- 단순 법령 정리가 아닌, **예규·판례·실무 사례 기반** 콘텐츠
- 사용자가 제공하는 PDF 서적에서 구체적 근거를 추출하여 인용
- "이런 제도가 있다" → "이 제도를 A사례에서 이렇게 활용했고, B판례에서 이런 쟁점이 있었다"

## 필수 포함 요소 (아티클당)

1. **예규/판례 최소 2개** — 번호, 쟁점, 결론 명시
2. **실제 사례 최소 1개** — 구체적 금액·조건 포함한 시뮬레이션
3. **실무 시사점** — 해당 사례가 독자에게 어떤 의미인지
4. **출처(references)** — frontmatter에 서적·법령·판례 명시

## 커스텀 블록 사용법

### :::case — 실무 사례 박스

```markdown
:::case

#### A법인 가업승계 사례

- 비상장법인, 제조업, 30년 경영
- 주식가액 500억원, 사업무관자산 비율 8%
- 가업상속공제 적용 시 절세효과: 약 230억원
  :::
```

### :::case-ruling — 예규/판례 박스 (⚖ 아이콘 자동 표시)

```markdown
:::case-ruling

#### 서면법규-2019-1234 (2019.05.15)

**쟁점:** 상속인이 등기이사로만 등재되어 있고 실제 근무하지 않은 경우 "가업 종사" 요건 충족 여부

**결론:** 단순 등기만으로는 부족. 출근기록·급여수령·업무보고서 등 실질적 종사 입증 필요

**시사점:** 상속인의 가업 종사는 형식이 아닌 실질로 판단되므로, 최소 2년 전부터 실제 업무 참여 기록을 체계적으로 남겨야 함
:::
```

### 기존 블록

- `:::callout` — 일반 강조 박스
- `:::callout-warn` — 경고/주의 박스
- `:::highlight` — 핵심 요약 박스

## PDF 활용 워크플로우

1. 사용자가 PDF 전달
2. Claude가 PDF 읽고 관련 예규·판례·사례 목록 정리
3. 아티클 주제에 맞는 것들을 선별하여 본문에 자연스럽게 배치
4. references에 서적 출처 명시

## 톤 & 스타일

- 전문가가 직접 쓴 것처럼 (이상준 세무사 명의)
- 법령 인용은 정확하게, 해설은 평이하게
- 독자 = 중소·중견기업 오너/경영진 (비전문가)
